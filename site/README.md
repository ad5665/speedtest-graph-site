### Graph/Chart Library
c3.js: http://c3js.org/

### CSV/JSON Parsing Library
PapaParse: http://papaparse.com/

### Only Requirement
You need a web server. I used python to initiate a web server.

```
python --version
Python 2.7.10
```

```
cd graphs-from-csv/
python -m SimpleHTTPServer
```

Visit http://localhost:8000 to see the website.
